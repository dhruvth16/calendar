# 📅 Calendar Notes App

A modern and responsive calendar-based notes application that allows users to select a date range and manage notes efficiently.

---

## 🚀 Features

- 📆 **Date Range Selection**  
  Select single or multiple dates using an interactive calendar.

- 📝 **Rich Text Notes Editor**  
  Create, edit, and delete notes with a powerful editor.

- 💾 **Local Storage Support**  
  Notes are stored in the browser using `localStorage`, ensuring persistence without a backend.

- 🎨 **Modern UI**  
  Styled using Tailwind CSS for a clean and responsive design.

---

## 🛠️ Tech Stack

- **Frontend:** React / Next.js
- **Styling:** Tailwind CSS
- **Calendar:** `react-day-picker` (customizable & supports range selection)
- **Editor:** `Tiptap` (rich text editor)
- **Storage:** Browser `localStorage`

---

## 📦 Installation & Setup

Follow these steps to run the project locally:

```bash
git clone <your-repo-url>
cd <your-project-folder>
npm install
npm run build
npm start
```
